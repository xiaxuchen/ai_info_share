公司核心概况速览
在正式分析前，我们先为这三家公司快速画像，让大家有个直观印象：
•
电气风电(688660)："海上风电急先锋" 。背靠上海电气集团，市场主要关
注其在海上风电的深厚底蕴和未来的盈利能力反转可能。需要警惕的是，
目前仍深陷亏损泥潭。
•
金风科技(002202)："全能型风电老大哥" 。A+H 股上市的老牌龙头，业务
覆盖风电全产业链。市场关注其庞大的风电场运营带来的稳定收益，以及
整机龙头的规模效应。
•
东方电气(600875)："能源装备多面手" 。不仅是风电玩家，更是火电、核
电、气电的全能选手。当前市场最热的"算电协同"和电力设备出海逻辑，
让它站在了聚光灯下。
多维度对比分析
接下来，我们将从六个核心维度，对这"三驾马车"进行深度解剖。
1. 所在行业
•
基本情况：三家公司均身处庞大的电力能源装备行业。其中，电气风电和
金风科技主业高度聚焦于风电，而东方电气则是覆盖了风电、火电、核
电、气电的综合能源装备巨头。
•
发展阶段：风电行业已进入平价上网的成熟期，甚至出现价格战的红海竞
争。但行业正经历深刻重构：一方面，海上风电向深远海迈进；另一方
面，行业与AI、算力等新物种产生奇妙"化学反应"——"算电协同"成为国
家战略，为绿色电力打开了全新的需求天花板。
•
竞争格局：风电整机制造商集中度不断提升，金风科技作为老牌龙头，市
场份额稳固。东方电气则在传统能源（煤电、气电）升级改造和核电领域
拥有寡头般的竞争优势。
•

正在彻底打开行业天花板。绿色燃料（如氢、氨）作为石油替代的战略价

2. 主营业务和商业模式
•
金风科技：商业模式最立体，是典型的"制造+服务+投资" 模式。除了卖风
机（风机及零部件销售占营收76.58%），它还通过自建风场卖电（风电场
开发），以及提供风电服务赚钱。这种"滚动开发"模式，能带来持续且稳
定的现金流。
•
东方电气：模式是典型的"高端装备+综合解决方案" 。营收来源多元且均
衡，不依赖单一能源品类。它的赚钱逻辑在于为整个电力系统提供最核
心、最壁垒高的发电设备，尤其是在核电主泵、燃气轮机等"大国重器"领
域。
•
电气风电：模式相对单一，高度依赖风机销售（销售产品占营收
92.6%），后市场服务占比很低。在风机价格战激烈的当下，这种"一条腿
走路"的模式使其抗风险能力较弱，这是目前最大的风险点。
3. 成长性分析（基于2025 年三季报）
•
营收增长率：电气风电( +35.96% ) > 金风科技( +34.34% ) > 东方电气
( +16.03% ) 。电气风电增速领跑，主要得益于低基数下的海上风电抢装或
项目集中交付。金风科技则展现了龙头稳健的扩张能力。
•
利润增长率：金风科技( +44.21% ) > 东方电气( +13.02% ) > 电气风电( -
53.51% ) 。
◦
金风科技：重点关注，其利润增速远超营收增速，显示出极强的盈利
能力和费用控制能力。第三季度单季净利润更是暴增170.64%，拐点
信号强烈。
◦
东方电气：增长稳健，但受传统能源业务周期性影响，增速较为平
稳。
◦
电气风电：需要警惕的是，虽然营收大增，但归母净利润亏损高达
7.32 亿元，且亏损同比扩大53.51%，"增收不增利"的现象极为严重，
主营业务的造血能力堪忧。

5. 竞争优势分析
•
金风科技：核心优势在于"规模和生态"。作为行业龙头，拥有最大的装机
存量，意味着其风电服务业务有巨大的增长空间。同时，庞大的风电场资
产既是利润来源，也是抵御整机周期性波动的"压舱石"。
•
东方电气：核心优势在于"技术纵深和全能"。在涉及国家能源安全的重大
装备领域拥有不可替代的地位。当前市场热炒的"算电协同"中，无论是火
电灵活性改造，还是核电作为基荷电源，东方电气都是绕不开的标的。重
点关注其出海逻辑，全球电网升级和"缺电"背景下，其高端电力装备有望
走向世界。
•
电气风电：核心优势在于"股东背景和特定市场"。依托上海电气集团，在
部分区域市场和海上风电领域仍有一定根基。但要将资源优势转化为利润
优势，还有很长的路要走。

结尾总结与风险提示
核心观点总结
•
金风科技：综合实力最强的优等生。不仅规模大，而且盈利结构优、增长
质量高。它已经摆脱了单纯"卖风机"的苦力活，成长为一家能提供稳定回
报的"新能源运营商"。重点关注其盈利能力能否持续改善。
•
东方电气：站在多重风口的全能选手。它既有传统能源升级的稳健，又有"
算电协同"和"出海"的想象空间。业绩增长稳健，确定性高，是能源转型浪
潮中的"卖铲人"。重点关注其核电、气电订单及出海进度。
•
电气风电：亟需证明自己的困境反转型选手。高增长背后是巨额亏损，说
明内部问题（成本、管理）尚未解决。虽然背靠大树，但打铁还需自身
硬。需要关注其毛利率何时转正，亏损何时能有效收窄。
未来需要关注的重要进展或指标
•
金风科技：风电场转让的进度和收益率；风机出口订单的获取情况。
•
东方电气：新增生效订单中，新能源（风电、核电）与海外业务的占比。
•
电气风电：单季度净利润能否转正；经营性现金流状况。
