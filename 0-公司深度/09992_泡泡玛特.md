一、基本情况及生意特性
1.1 公司概况
泡泡玛特国际集团有限公司（股票代码：09992.HK）创立于2010年，2020年12月在港交所上市，是一家以IP运营
为核心的潮流文化娱乐集团。公司主要从事潮流玩具的产品设计与开发、品牌运营及全渠道销售，覆盖手办、毛绒
玩具、MEGA收藏品、衍生品等多个品类。
治理结构：创始人王宁为董事会主席，管理层多为早期创业团队，股权相对集中。2025年末公司拥有员工约1.09万
人（同比+64.9%），其中设计师及行政人员2194人。
上市与资本市场：总股本约134.1亿股，2026年4月28日收盘价153.2港元，对应总市值约2054亿港元（约1850亿人
民币）。股价从2025年8月历史高点339.8港元累计回调54.9%，市值蒸发逾2000亿港元。

二、核心逻辑及业务分析
2.1 市场空间
全球潮玩及 IP 衍生品市场规模：全球潮流玩具市场（blind box + 手办 + 毛绒收藏品）规模估算约200-300亿美元，
中国市场约500-800亿人民币。根据2025年上半年数据，泡泡玛特以约139亿元人民币的半年玩具收入，已超越迪士
尼和万代南梦宫，跃居全球玩具品牌收入第二位，仅次于乐高。从这个角度看，泡泡玛特的天花板已经不是"潮玩"
，而是"全球IP消费品"——对标乐高（年营收约700亿丹麦克朗≈700亿人民币），泡泡玛特2025年371亿元收入仍有
翻倍空间。
IP 消费品的新范式：LABUBU的成功证明，优质IP + 稀缺性营销 + DTC渠道 + 全球化运营，可以实现远超传统玩具
的消费爆发力。LABUBU从泰国爆火到席卷全球多个市场，体现了IP文化穿透力的新高度。
渗透率逻辑：泡泡玛特当前在全球20个国家有直营布局，全球玩具/收藏品市场渗透率仍然极低，尤其美洲（2025年
营收+748%）和欧洲（+506%）仍处于早期扩张阶段，类比中国市场的早期高速成长期。

泡泡玛特的护城河：
1. IP 矩阵积累：93个运营IP（12个自有、25个独家、56个非独家），17个IP收入过亿，头部IP持续孵化更新，
降低单IP依赖风险（尽管LABUBU当前占比仍高）。
2. DTC 直营能力：630家直营门店覆盖20国，门店既是销售渠道也是品牌展示和粉丝互动空间，门店运营效率高
（单店坪效突出）。
3. 数字化运营：自研APP及官网收入快速增长（2025年海外线上43.54亿元，+1094.9%），TikTok等社交电商渠
道深度布局。
4. 供应链柔性：核心供应商集中采购+柔性生产，能够快速响应爆款需求，同时保持产品稀缺性。
5. 管理层执行力和先发优势：泡泡玛特在海外市场，特别是东南亚已建立显著的品牌认知和门店网络，后来者难
以快速复制。
2.3 成长驱动
核心成长逻辑：
① 海外市场持续扩张：海外业务2025年同比+292%，美洲+748%、欧洲+506%，东南亚+619%（2024年数据），
均处于加速阶段。2026年管理层指引海外新开门店超过2025年，美洲市场从"早期"向"规模化"跨越，客单价约为中
国市场的2-2.5倍（美洲最高），量价齐升逻辑清晰。
② 品类扩张：毛绒产品爆发（+560%）已证明泡泡玛特的品类复制能力；2026年计划推出家电产品线和甜品业务，
北京乐园扩建，从玩具公司向生活方式品牌延伸，打开变现天花板。
③ 门店扩张：2025年净增109家门店，2026年计划维持积极扩张节奏。DTC直营模式使单店模型高度可控，随规模
扩大费用率持续优化（销售费用率从28%降至21.8%）。
④ IP 持续孵化：2025年17个IP过亿，星星人等新IP增长超1600%，多IP矩阵逐步稀释LABUBU单一依赖风险；公
司持续签约新设计师、拓展授权IP，保持IP库活力。
盈利改善空间：毛利率从64%→72.1%（2023-2025），仍有提升空间，主要来自：海外高客单价市场占比提升（海
外客单约国内2-2.5倍）、规模效应带来的采购成本下降、自产比例提升。费用率方面，销售费用率已从28%降至21
.8%，管理费用率从10%降至4.8%，规模效应尚未完全释放。
三、主要风险
3.1 IP 集中度风险
LABUBU单一IP 2025年收入141.61亿元，占总收入38.1%。若LABUBU热度下降（如2025年8月以后的阶段性回调
已有所体现），公司收入增速将面临断崖式回落。市场对LABUBU能否持续支撑增长存在分歧——2025年财报后股
价单日暴跌22.51%，正是这种担忧的集中反映。
3.2 高增长可持续性风险
2025年营收+184.7%、利润+308.8%，是在LABUBU全球爆发 + 门店快速扩张 + 品类爆款三重叠加下的结果。202
6年（管理层指引+20%）增速环比大幅放缓已是确定性事件，"再定基"之年能否平滑过渡，存在较大不确定性。
3.3 存货与现金流风险
存货余额54.73亿元（+259%），周转天数从102天升至123天。随着LABUBU热度阶段性回落，若存货出现滞销，
可能面临较大减值压力。自由现金流状况需重点跟踪。
3.4 地缘政治与汇率风险

---

海外业务（占43.8%）覆盖东南亚、欧美等不同政策环境，面临进口关税、文化合规等潜在摩擦；美元、东南亚各
国货币汇率波动可能影响利润折算。
3.5 竞争加剧风险
随着泡泡玛特商业模式的成功，国内可能出现更多模仿者（如名创优品旗下TOPTOY等）；海外市场，万代、迪士
尼等传统巨头也可能加大对"设计师IP + 盲盒"模式的关注和投入。
四、其他重要问题
4.1 多元化扩张的边界
泡泡玛特正从玩具向家电、甜品、主题乐园等延伸。多元化有助于打开天花板，但也存在资源分散、品牌稀释的风
险。家电和甜品与核心IP业务的协同效应尚待验证，2026年是关键观察期。
4.2 分红政策
公司历史上分红较少，主要将利润投入扩张。2025年现金储备137.75亿元，经营性现金流充裕但分红率仍低。对于
追求现金回报的投资者，泡泡玛特并非理想标的。
4.3 股价大幅回调的背景
股价从2025年8月高点339.8港元跌至153.2港元（-54.9%），主要反映：①
高位估值泡沫消化（当时PE一度超80倍）；② 
市场对高增长可持续性的担忧；③
财报后外资机构净卖出。当前PE约14-15倍，已从"极度高估"回归至"合理偏低"区间。
