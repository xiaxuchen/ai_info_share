晶盛机电（300316）
作者：开卷有财
一、基本情况及生意特性
公司是全球光伏设备龙头，并成功将能力延展至半导体装备与材料领域，形成了"
装备+材料"双轮驱动的平台化布局。其生意特性具有强周期性（与下游资本开支高
度相关）和高技术壁垒（晶体生长、精密加工等核心工艺）双重特征。当前，公
司正利用其在晶体生长与材料制备方面的共性技术，积极应对光伏周期，并开拓碳
化硅、太空光伏等新兴领域。
二、核心逻辑及业务分析：三大增长曲线
公司价值已由单一周期驱动，转变为由传统业务复苏、第二曲线放量、前沿领域
赋能三重逻辑共同驱动。
第一曲线（光伏设备）：周期等待复苏，技术迭代蓄力
现状：行业产能过剩导致需求阶段性萎缩，公司设备订单下滑。但光伏作为能源
转型主力，长期装机需求明确。
未来驱动：N 型技术替代、薄片化、去银化等降本增效技术迭代，将催生存量设
备的升级替换需求。公司作为可提供电池、组件整线解决方案的龙头，将率先受
益。
第二曲线（碳化硅材料）：成长进行时，关键在8 英寸量产
市场：新能源汽车、能源是核心驱动力，从6 英寸向8 英寸衬底升级是降低成
本、提升竞争力的产业共识。
公司优势：独特的"设备研发+材料量产"一体化模式，通过自身产线验证反哺设备
优化，构建了技术和成本的双重护城河。90 万片（尤其是8 英寸）产能规划是
国内最激进的布局之一，成败关键在于良率爬坡和客户认证速度。
第三曲线（太空光伏赋能）：前沿"期权"，技术平台的延伸验证
市场本质：太空光伏是为满足低轨卫星星座及未来太空设施能源需求，对光伏技
术进行的高可靠、轻量化、抗辐射的极致场景化应用。
公司角色：晶盛机电并非直接生产太空电池，而是作为上游高端装备与材料工艺
的赋能者。其价值在于为下游电池/组件厂商生产满足太空级要求的"超薄硅片"、"
高品质晶体"提供关键设备（如低氧单晶炉、精密切片机）和工艺解决方案。
技术路径关联：当前最具经济性的太空电池技术路径是超薄硅基异质结（HJT），
这正是公司传统优势领域。未来若钙钛矿/叠层电池取得突破，公司已布局的
PECVD、ALD 等薄膜沉积设备同样具备切入潜力。

三、主要风险
1.
行业周期性下行风险：光伏行业复苏若慢于预期，将持续压制公司主要收入和
利润。
2.
新业务产业化不及预期风险：碳化硅8 英寸衬底量产良率、成本及客户认证
进度是核心观察点，若延迟将影响第二曲线成长叙事。太空光伏的远期大规模
部署存在技术可行性与经济性挑战，可能长期无法落地。
3.
技术路线摇摆风险：光伏或半导体材料技术发生颠覆性变革，或太空光伏跳过
硅基路线，可能削弱公司现有技术储备的价值。
4.
市场竞争加剧风险：各赛道参与者均加速扩产，可能导致价格战，侵蚀利润。
