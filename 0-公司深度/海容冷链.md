一、基本情况及生意特性
1.1 业务范围与产品结构
海容冷链成立于2006年，总部位于山东青岛，主营商用冷链设备的研发、生产和销售。公司产品按
功能分为四大类：
•
商用冷冻展示柜：核心产品，面向冰淇淋、速冻食品等品牌商，用于商超、便利店等场景的冷冻
食品陈列，单台价格根据规格在1,000-5,000元不等。
•
商用冷藏展示柜：近年来快速增长的品类，面向饮料、乳制品、啤酒等品牌商，用于冷藏饮品陈
列，与冷冻柜同属陈列柜大类但单价略低。
•
商超展示柜：面向连锁超市、便利店的冷链陈列设备，包括冷鲜柜、制冰机等。
•
商用智能售货柜：自动售货机的一种，支持冷藏食品的无人零售，目前体量较小但增速较快。
2024年收入结构参考：冷冻展示柜约占55%，冷藏展示柜约25%，商超柜约12%，智能售货柜及其
他约8%。冷冻柜占比最高但占比呈下降趋势，冷藏柜和出口产品占比持续提升。
1.2 销售模式与产业链地位
公司采用B2B定制化直销模式，客户为食品饮料、乳制品等品牌商。公司按客户要求设计、定制冷柜
，然后通过投标/议价方式获取订单。这一模式决定了：
•
客户认证周期长：品牌商对冷柜的制冷效率、能耗、外观、可靠性有严格要求，认证周期通常6-1
8个月，一旦进入供应链则粘性较强。
•
应收账款规模适中：以大客户为主，账期管理相对规范。
•
成本结构：直接材料（压缩机、钢板、制冷剂等）约占成本70%，制造成本约20%，其余为物流
和售后成本。
1.3 股权结构与分红
公司实际控制人为管理层及核心员工，股权结构相对稳定。上市以来（2018年）累计分红约6-7亿元
，年度分红占净利润比例约30-40%，分红政策较为稳定。2025年拟派发现金红利（具体金额待年报
正式披露，按惯例约每10股派2-3元）。

二、核心逻辑及业务分析
2.1 市场空间
国内商用展示柜市场约150-200亿元。 其中商用冷冻展示柜（冰淇淋+速冻食品）市场约80-100亿，
商用冷藏展示柜（饮料+乳制品）市场约60-80亿，商超柜及智能售货柜合计约30-50亿。
增速方面，国内商用冷链设备市场过去5年复合增速约8-10%，主要由以下因素驱动：
•
冷链渗透率提升：国内人均冷链设备保有量仍低于欧美日等发达国家，尤其是二三线城市和县域
市场便利店扩张带来增量需求。
•
消费升级：消费者对冷鲜食品品质要求提升，品牌商愿意投入更多冷柜以提升终端曝光率。
•
出口增长：中国商用冷柜制造商凭借成本优势（比欧美同类产品便宜30-50%），加速抢占全球市
场份额，出口增速过去三年超过30%/年。

海容冷链的国内目标市场约50-80亿，公司目前国内收入约25亿，市占率约30-40%（冷冻柜品类市占

2.2 竞争格局
国内商用展示柜行业竞争格局呈现"一超多弱"特征，海容是绝对龙头，第二梯队包括澳柯玛、冰山集
团等，但规模和技术实力差距明显。
海容的核心竞争优势：
1.
规模效应：年产各类商用冷柜超过200万台，规模带来原材料采购成本优势和生产线自动化程度高。
2. 客户壁垒：伊利、蒙牛、康师傅、农夫山泉、统一、雀巢、和路雪、可口可乐、百事可乐等国内外
一线品牌均为长期客户。这些品牌对供应商认证严格，进入壁垒高，一旦认证通过不会轻易更换。
3. 产品完整性：从冷冻柜到冷藏柜到智能售货柜，产品线覆盖面广，可以提供一站式解决方案。
4. 出口能力：已建立覆盖全球50多个国家的销售网络，出口业务增速显著快于国内。
主要竞争对手：
澳柯玛（AUCMA）在医用冷链和家用冷柜领域较强，商用展示柜规模较小；Jiangsu
Jilong等区域性厂商主要在特定区域或细分品类与海容竞争。
2.3 成长驱动因素
驱动一：出口业务高速增长
2025年上半年，公司出口收入同比增长约58%，是增长最快的业务板块。出口产品以商用冷冻柜和
冷藏柜为主，主要销往东南亚、中东、南美和欧洲。核心逻辑是中国制造业的成本竞争力（原材料成
本、劳动力成本均低于欧美竞争对手），且质量已通过可口可乐、百事可乐等全球品牌认证。
公司正在建设海外生产基地（东南亚），主要目的是规避部分国家的贸易壁垒（如东南亚国家对进口
中国冷柜的反倾销调查），同时降低物流成本。东南亚工厂投产后，出口业务有望进一步加速。预计
2026年出口占比从目前约20%提升至25-30%。
驱动二：冷藏展示柜持续放量
冷藏柜面向饮料和乳制品客户，产品单价略高于冷冻柜，客户群体与冷冻柜高度重合，因此可以复用
现有渠道。公司冷藏柜业务经过2022-2023年的客户培育期，2024-2025年进入快速放量阶段。该品
类毛利率略低于冷冻柜，但收入规模扩大带来的产能利用率提升可以部分对冲。
驱动三：新客户持续开拓
除已有的乳制品和冰淇淋客户外，公司积极开拓啤酒、预制菜、社区团购等新客户。啤酒品牌（如雪
花、青啤）在夏季的冷藏陈列柜需求量大，是潜在的增量来源。预制菜和社区团购的冷链配送柜需求
则是更长线的增长机会。

三、主要风险
3.1 下游需求周期性风险
公司约60%以上收入来自冰淇淋、饮料、乳制品等快消品，这类产品需求受天气和消费周期影响较大
。2024年收入下滑11.7%就是明证——暖冬和下游渠道去库存导致冷柜需求骤降。这种周期性波动意
味着公司业绩不可能持续线性增长，在某些年份可能出现营收和利润的双降。
3.2 原材料成本波动风险
压缩机（占成本的约20%）、冷轧钢板（占约25%）、制冷剂等主要原材料价格与大宗商品价格高度
相关。2021-2022年钢材价格大幅上涨对公司毛利率造成压力。若原材料再次进入涨价周期，而公司
无法及时向下游传导，将压缩利润空间。
3.3 客户集中度风险
前十大客户占收入比例约40-50%，其中伊利、蒙牛两大乳制品巨头合计占比估计超过20%。若这两
家客户因战略调整降低冷柜投放密度，或将供应商份额向竞争对手转移，将对公司造成显著收入冲击
。
3.4 汇率风险
出口收入占比约20%，产品销往全球50多个国家，以美元结算为主。人民币升值将直接压缩出口毛利
率。公司虽通过远期结汇等工具进行部分对冲，但汇率波动仍构成不可忽视的利润风险。
3.5 海外工厂建设风险
公司正在东南亚建设海外生产基地，涉及土地购置、厂房建设、设备采购等，资本开支较大。建设进
度可能受当地政策、审批流程、汇率波动影响。若海外工厂投产延迟或运营初期亏损，将拖累整体业
绩。

四、其他重要问题
4.1 ROE偏低的原因分析
公司ROE长期维持在9-10%，在制造业中属于中等偏低水平。拆解来看：净利率11-13%（尚可）但
总资产周转率偏低（约0.5次/年），权益乘数约1.6-1.8（负债率适中偏低）。周转率低是因为资产偏
重（冷柜生产线、模具设备投资较大），且订单式生产导致产能利用率随下游需求波动。ROE要想
显著提升，需要收入增速持续高于资产增速，或者提高负债率——目前看，两者的可能性都有限，R
OE 10%附近可能是较长期的均衡水平。
4.2 与竞争对手的关键差异
海容冷链的核心壁垒不在于技术唯一性（制冷技术较为成熟），而在于与品牌商长期合作积累的信任
和认证资质。一台冷柜看似简单，但品牌商对能耗（影响门店电费）、制冷均匀性（影响食品保鲜）
、故障率（影响品牌形象）的综合要求，使得新进入者需要2-3年才能完成认证，这本身就是天然护
城河。
4.3 分红与再投资平衡
公司年CAPEX约3亿，OCF约4-7亿，自由现金流转正但规模有限。考虑到东南亚工厂扩建计划，未
来2-3年CAPEX可能上升至4-5亿/年，分红能力受限。分红率预计维持30-40%的合理水平，不宜预期
大幅提升。

## 关键要点

**投资逻辑**：商用冷链设备绝对龙头，出口业务高速增长和冷藏柜新品类放量是核心弹性；若出口收入占比从20%提升至35-40%，冷藏柜收入超过冷冻柜，成长天花板将进一步打开。

**核心竞争力**：年产各类商用冷柜超200万台的规模效应；与伊利、蒙牛、可口可乐等一线品牌长期合作形成的客户认证壁垒；覆盖全球50多个国家的出口网络。

**主要风险**：下游冰淇淋、饮料等快消品需求受天气和消费周期影响大（2024年营收同比下降11.7%）；原材料（压缩机、钢板）成本与大宗商品价格高度相关；前十大客户占收入40-50%，集中度较高。

**财务亮点**：2025年营收33.51亿元（同比+18.4%），经营现金流净额6.83亿远超净利润3.87亿；2026年Q1营收8.37亿，净利率12.20%。
