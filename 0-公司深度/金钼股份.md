一、基本情况及生意特性
1.1 业务范围
金钼股份是典型的"矿到材"一体化企业，业务覆盖钼全产业链：上游钼矿采选、中游冶炼（钼炉料、钼化
工）、下游深加工（钼金属、钼靶材）。主要产品分四大板块：
• 钼炉料（钼铁、氧化钼）：公司的基本盘，收入占比约65%-70%，是利润的主要来源
• 钼化工（钼酸铵、钼酸钠等）：占比约10%-12%
• 钼金属（钼粉、钼条、钼丝）：占比约12%-15%
• 深加工产品（半导体级高纯钼靶材等）：占比约5%-8%，但增速最快、想象空间最大
公司销售以国内市场为主，客户集中在钢铁、石化、航空航天、半导体等领域的龙头企业，直销为主。
1.2 产业链地位
说金钼股份是"国内钼王"不为过。核心矿区的资源自给率超过95%，基本不用在外头买矿，这在国内有色
企业里相当少见。

二、核心逻辑及业务分析
2.1 市场空间
钼的总体需求：
2026年全球钼需求预计31.7万吨，同比增长5%-6%。需求结构：钢铁（约75%，特钢和不锈钢为主）、
新能源（光伏靶材、风电、新能源车用钼快速增长）、化工催化剂、军工/核电/半导体。
供给端2026年仍存在约4.43万吨的缺口（2025年为3.64万吨），处于连续第三年供不应求状态。
半导体钼靶材市场：
2025年全球半导体靶材市场规模约48.3亿美元，2026-2028年复合增速14.2%。钼靶材作为半导体薄膜沉
积工艺的关键材料（用于芯片电极和互联层），在AI芯片、先进制程中需求增长尤其迅猛。2026年钨、钼等小
金属靶材价格暴涨60%-70%，3nm/5nm高端靶材供需缺口已超30%。
2.2 竞争格局及优势
上游钼矿领域：
全球钼矿主要集中在中国、美国、南美。国内钼矿竞争格局相对集中，金钼股份（市占率27%-28%）是
绝对龙头，洛阳钼业资源量也很大但以铜钴为主。金钼的核心优势在于：自有矿山、资源自给率95%+，成本
端具备显著优势，在钼价下行周期里有很强的"安全垫"。
下游钼靶材领域：
这是金钼股份最有想象力的赛道。
技术突破实锤：公司已实现5N级（99.9995%）高纯钼靶材规模化量产，是国内唯一实现该级别量产的企
业。5N级钼靶材已通过台积电认证，并与中芯国际签有长单，同时供货京东方等面板龙头。
这个赛道的壁垒很高。高纯钼靶材需要从矿石到最终产品的全产业链能力，中间环节任何一个出问题都做
不出来。金钼恰恰是全产业链布局，全球六大原生钼矿掌控者之一，别人想复制很难。
半导体级钼材国内市占率33.6%，国内第一。成熟制程靶材国产化率已达60%-70%，先进制程（7nm以下）
也突破15%-20%，金钼是国产替代的核心受益标的。
2.3 成长驱动
驱动一：钼价弹性
这是最核心的短期驱动。2026年钼价走势强劲：
• 钼精矿Q1均价4238元/吨度，同比上涨20.6%
• 5月中旬钼铁报价已达32.5-33.2万元/吨，较年初上涨32.5%
• 钼精矿5月成交价创下550元/吨度的年度新高
钼价上涨直接提升公司冶炼环节的利润率。金钼的自有矿成本相对固定，钼价每涨100元/吨度，公司利润
弹性非常大。以年产5万吨钼精矿估算，钼价每变动100元/吨度，年化利润变动约5亿元，弹性相当可观。

钼价上涨的驱动因素包括：全球供给缺口持续扩大（连续三年供不应求）、新能源需求增量、中东地缘冲
突导致伊朗钼出口中断（2025年伊朗产量约3300吨金属量）、以及全球绿色转型对特钢需求的拉动。
驱动二：半导体钼靶材放量
这是中长期最重要的成长逻辑。5N级钼靶材通过台积电认证+绑定中芯国际长单，相当于拿到了全球最头
部客户的入场券。随着国产替代加速推进和AI芯片对先进制程靶材需求爆发，钼靶材业务的收入和利润贡献将
持续提升。
目前钼靶材业务占比还只有5%-8%，但增速远超传统冶炼业务。如果未来3-5年占比提升到15%-20%，对
公司整体利润率也会有拉动作用（靶材业务毛利率预计高于冶炼业务）。
驱动三：沙坪沟钼矿
沙坪沟钼矿是全球最大的单体钼矿，储量惊人。2026年1月15日，金钼股份以17.31亿元收购紫金矿业持
有的安徽金沙钼业24%股权，持股比例从10%提升到34%。2026年1月，沙坪沟钼矿采选项目已获安徽省自然
资源厅审批通过，建设即将全面启动。
这个矿的意义在于：投产后将大幅增加金钼的钼精矿产能，进一步巩固全球龙头地位。按照沙坪沟的设计
产能，完全达产后金钼的钼精矿产能可能接近翻倍。当然，投产时间还有不确定性，预计需要3-5年的建设周
期。
三、主要风险
3.1 钼价周期性波动风险
这是最大的风险，没有之一。钼是典型的周期性商品，价格波动剧烈。回顾历史：
• 2016-2017年：钼价低迷，公司ROE只有0.4%-0.8%，利润极薄
• 2020年：受疫情影响，营收下降，ROE仅1.4%
• 2023年：钼价暴涨，ROE飙升至21.8%
周期股的特点就是"三年不开张，开张吃三年"。如果全球钢铁需求放缓、新能源增速不及预期，或者主要
钼矿产能集中释放，钼价可能再次进入下行通道。公司2024年利润略低于2023年，就是钼价高位震荡的体现。
不过要看到，金钼的"安全垫"比一般有色企业厚实。95%+的资源自给率意味着在钼价下行周期里，成本端
的压力相对可控，不太可能出现亏损，只是利润会大幅缩水。
3.2 半导体钼靶材认证进度不及预期
半导体行业对靶材的认证周期很长，通常需要1-2年。虽然公司已经通过了台积电认证并与中芯国际签单
，但要大规模放量仍面临：先进制程（3nm/5nm）靶材的技术门槛更高，与国际巨头（日本JX金属、美国
Honeywell）在纯度和大尺寸稳定性上还有差距。如果认证进度不及预期，钼靶材的放量节奏会被拖慢。
3.3 沙坪沟钼矿投产进度
沙坪沟钼矿虽然审批已通过，但大型矿山从建设到完全达产通常需要3-5年。期间可能遇到：环评审批、
建设资金、技术攻关等各种问题。如果投产进度大幅推迟，公司产能扩张的故事就会打折扣。
此外，沙坪沟投产后可能带来钼供给的增加，中长期反而可能压制钼价。这是一个"双刃剑"。

四、其他重要问题
4.1 分红政策
金钼股份近三年分红非常稳定：2023-2025连续三年每10股派4元（含税），年分红约12.9亿元，股利支
付率约40%-45%。上市以来累计分红17次，总额超过116亿元。
在有色金属行业里，这个分红力度相当厚道。40%+的股利支付率说明公司不是那种只顾扩产不顾股东回
报的企业。考虑到公司资产负债率极低（12%左右），未来分红比例还有提升空间。
4.2 资本开支计划
2025年金钼集团（母公司层面）固定资产投资7.58亿元。上市公司的资本开支主要用于：
• 钼靶材产线扩建和升级
• 矿山设备更新和维护
• 环保设施改造
以公司目前的自由现金流水平（年经营现金流约20-37亿元），完全有能力在不增加负债的情况下支撑资
本开支和分红。沙坪沟钼矿的投资主要通过参股方式完成，对母公司现金流压力不大。

5.2 盈利能力评估
不看估值数字，用DCF逻辑来理解金钼的盈利能力：
核心判断：金钼股份是一个"下有底、上有弹性"的盈利模式。
底部：即便钼价跌到成本线附近，公司凭借自有矿山（采选成本低于外购矿石）、低负债（几乎没有财务
费用）以及稳健的分红政策，大概率能维持正利润，不至于亏损。2016-2017年钼价低谷期，公司虽然利润极
薄，但从未亏损。

5.3 安全边际与验证指标
跟踪金钼股份，需要持续关注以下核心指标：
1. 钼价走势（每周跟踪）：钼精矿（45%）价格、钼铁价格，这是影响业绩的最直接变量
2. 钼靶材认证和放量进展（季度跟踪）：是否通过更多头部客户认证、靶材收入增速
3. 沙坪沟钼矿建设进度（季度跟踪）：环评、建设、投产时间表
4. 公司季度毛利率和净利率变化：直接反映钼价对公司利润的传导效率
5. 全球钼供需平衡表：跟踪国际钼协会（IMOA）数据，关注供需缺口变化
5.4 最终结论
金钼股份是一个"好生意、好公司、需要好价格"的典型标的。
好生意：钼作为战略金属，在绿色转型和半导体领域的需求结构性增长，供需连续三年缺口扩大，行业景
气度处于上升周期。
好公司：国内钼行业绝对龙头，资源自给率95%+，全产业链一体化，5N级钼靶材实现国产突破并打入台
积电供应链，财务稳健、分红厚道。
成长逻辑清晰：短期看钼价弹性（2026年钼价涨幅超30%），中期看钼靶材国产替代放量，长期看沙坪沟
钼矿投产后产能翻倍。
主要风险：周期性是最大的不确定性。钼价不可能永远涨，一旦进入下行周期，利润会大幅波动。
综合来看，金钼股份适合作为有色周期成长股的配置标的，但需要严格跟踪钼价走势和钼靶材业务进展。
在钼价高位区间追高需要谨慎，在钼价因短期因素回调时可能是更好的介入时机。

## 关键要点

**投资逻辑**：钼价上行周期（2026年钼精矿均价同比+20.6%）带来利润弹性；5N级高纯钼靶材通过台积电认证并绑定中芯国际长单，半导体靶材国产替代放量；沙坪沟钼矿权益提升至34%，投产后产能有望翻倍。

**核心竞争力**：国内钼行业绝对龙头（市占率27%-28%），资源自给率超95%；全球唯一覆盖"矿到靶材"全产业链的企业，5N级钼靶材国内唯一规模量产；资产负债率仅11%-13%，年经营现金流20-37亿元，财务稳健。

**主要风险**：钼价周期波动为核心风险，2016-2017年低谷期ROE仅0.4%-0.8%；半导体靶材先进制程（3nm/5nm）与国际巨头仍有差距，放量节奏存不确定性；沙坪沟钼矿投产后中长期可能增加供给、压制钼价。

**财务亮点**：2025年营收138.34亿元，归母净利润31.55亿元，毛利率41.1%，净利率25.4%，ROE 17.6%；连续三年每10股派4元，股利支付率40%-45%，上市以来累计分红超116亿元。

**估值参考**：综合评分4.0/5，定性为优质周期成长股；钼价每变动100元/吨度，年化利润变动约5亿元；当前钼价下年化归母净利润30-35亿元为合理区间。
