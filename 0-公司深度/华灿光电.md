华灿光电（300323）
一、基本情况及生意特性
1. 业务范围
华灿光电是国内第二大LED芯片供应商（仅次于三安光电），核心产品为LED外延片和芯片，覆盖蓝绿光LED、红黄光LE
D、紫外/红外LED等。京东方（000725）于2023年入主，持股超20%，为公司控股股东。2024年更名"京东方华灿光电"，定
位从独立的LED芯片厂转向京东方的"半导体光电平台"。
近年新增方向：Micro 
LED（微米级LED显示芯片）、玻璃基板封装用Micro 
LED固晶设备、光互连（optical
interconnect）芯片。通过定增融资约10亿元投向Micro LED晶圆制造和封装测试项目（金湾一期）。
2. 业务结构
2025年营收54.08亿元（+31.07%），结构如下： 
- 
LED芯片：收入约30-35亿，毛利率8-12%，占营收60%+ 
-
贵金属回收：收入约15-20亿（LED芯片制造过程中使用贵金属催化剂，回收后出售），毛利率5-8% 
- 
Micro
LED及其他：收入极小，主要还在研发和送样阶段
2026Q1营收暴增至22.21亿元（+109.74%），归母净利润5067万元，实现了自2023年以来的首次单季盈利。拆开来看，
增长大头来自贵金属回收业务量价齐升（金价上涨+回收量增），芯片主业改善幅度有限。
3. 产业链地位
上游：蓝宝石衬底、MO源（金属有机化合物）、高纯气体等。供应商集中度较高，但对龙头芯片厂的议价能力不强。
下游：封装厂（国星光电、瑞丰光电等）+面板背光模组（京东方、TCL等）+照明终端。京东方的入主解决了最大痛点—
—客户。华灿的LED芯片直接进入京东方背光模组供应链，订单稳定性大幅提升。
成本结构：相对重资产。MOCVD外延设备单台价值数百万元，折旧占成本约15-20%。贵金属耗材是另一大成本项。202
6Q1毛利率11.27%，比一年前翻了一倍多，主要是产能利用率提升摊薄了固定成本。
4. 股东背景
京东方持股超20%，为控股股东。2026年4月公告京东方拟现金认购定增不超过9.98亿元，若实施将进一步提升持股比例
。机构持仓以产业资本为主，公募基金持仓较少。

二、核心逻辑及业务分析
1. 市场空间
LED芯片：全球市场规模约200-250亿美元，年增速3-5%。已经是成熟市场。竞争格局趋于寡头化——三安光电国内市占
约30-35%，华灿约15-20%，其余分散在兆驰、乾照等手中。
Micro 
LED：全球市场预期2030年约200-300亿美元（Apple
Watch/AR眼镜/车载显示/大尺寸商显等场景）。目前产业化处于"黎明前夜"——Apple 
Watch原计划2025年用Micro
LED，推迟了；AR眼镜是更确定的方向（Meta、苹果都在推）。华灿的Micro
LED芯片+固晶设备布局，本质是在押注这个百亿美元级别的新市场。
光互连：AI算力芯片间用光代替电传输数据，是解决带宽瓶颈的前沿技术。如果光互连芯片（基于玻璃基板+Micro LED阵
列）能在2028-2030年进入数据中心，市场空间可能是千亿级别。这个方向现在处于"从0到1"的实验室阶段，不确定性极大。
京东方的康宁合作提到"光互连相关应用"，间接为华灿打开了想象空间。
2. 竞争格局
LED芯片：三安光电遥遥领先——营收150亿+、毛利率20-25%、净利率8-10%。华灿在体量、盈利能力和技术储备上都
有明显差距。但京东方的入主改变了竞争生态——华灿有了"保底客户"，不再需要在公开市场上与三安正面厮杀。
Micro 
LED：这是一个全新的赛道，全球还没有明确的赢家。台湾錼创（PlayNitride）、英国Plessey（Facebook收购）
、法国Aledia等是海外主要玩家。国内三安+华灿+乾照+辰显光电都在布局。华灿的优势在于京东方和辰显光电的生态协同，
劣势在于资金实力弱于三安。
3. 成长驱动
• 短期（2026-2027）：LED芯片主业恢复盈利（产能利用率提升+产品结构优化），贵金属回收贡献稳定利润。定增落地缓解
资金压力。 - 中期（2027-2029）：Micro
LED在AR眼镜/车载显示/大尺寸商显领域的量产突破。如果苹果或Meta在消费级AR眼镜上采用Micro
LED，华灿作为上游芯片供应商可能获得超预期订单。 -
长期（2029+）：光互连芯片商业化（这是最大的"彩票"——如果中了回报巨大，不中也不意外）。
4. 核心投资命题
华灿是一个"业绩反转+期权"的故事。业绩反转指的是LED芯片主业从连续3年亏损走向盈利（2026Q1已出现拐点信号）
；期权指的是Micro LED和光互连这两个"可能改变公司命运"的前沿方向。
三、主要风险
Micro 
LED产业化不及预期。这是最大的风险，也是最核心的不确定性。Micro
LED从样品到量产到大规模商用，时间线高度不确定。如果Micro
LED的商业化推迟到2030年以后，华灿当前市价中隐含的Micro
LED期权价值会大幅缩水。事实上，当前200亿左右市值里，估计有100亿以上是对Micro
LED未来的预期——这个预期的"证伪"成本很高。
贵金属回收利润不可持续。2026Q1利润暴增很大程度是靠金价上涨驱动的贵金属回收业务。金价波动性大，如果金价下
跌20-30%，这部分利润会显著缩水。而且回收业务的利润率（5-8%）远低于芯片主业，未来不应该成为利润的主要来源。
与京东方的绑定风险。京东方的深度赋能是把双刃剑——既有订单稳定性，也有"被整合"风险。如果未来京东方选择自建L
ED芯片产能或者转向其他供应商，华灿的独立性会面临挑战。
财务压力。2025年仍亏损4.38亿，账面净资产约65亿。如果Micro
LED的研发和产能建设持续烧钱，而LED芯片主业盈利改善不够快，现金流压力会在2027-2028年体现。

## 关键要点

**投资逻辑**: 业绩反转+前沿技术期权：LED芯片主业从连续3年亏损走向盈利（2026Q1扭亏），京东方入主提供订单稳定性；Micro LED（AR眼镜/车载/商显）和光互连芯片是两大潜在爆发期权。

**核心竞争力**: 国内第二大LED芯片厂（市占15-20%），京东方控股深度绑定客户（背光模组供应链），Micro LED芯片+固晶设备双布局，已向辰显光电交付玻璃基板Micro LED固晶设备。

**主要风险**: Micro LED产业化时间线高度不确定（Apple Watch已推迟），200亿市值中估计100亿+是对Micro LED的预期，金价驱动的贵金属回收利润不可持续，与京东方的"被整合"风险。

**财务亮点**: 2026Q1单季首度扭亏（净利5067万/毛利率11.27%），经营现金流（~5亿）远好于净利（-4.38亿），定增落地将改善资产负债结构。

**估值参考**: 分部估值：LED基本盘（2026净利2亿,PE 20-25x）40-50亿 + Micro LED期权30-60亿 = 合理70-110亿。当前约200亿市值已大幅溢价。
