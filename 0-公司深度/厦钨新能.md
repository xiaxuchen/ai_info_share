厦钨新能（688778）
作者：开卷有财
一、基本情况及生意特性
1.
业务范围

核心产品线：公司是全球锂电正极材料领域的领军企业，核心产品包括钴
酸锂、三元材料（含中镍高电压、高镍、NL 材料）、磷酸铁锂以及氢能
材料（储氢合金粉）。

子公司分工：母公司厦门钨业提供强大的资源和技术支持。公司本部及各
基地（如海璟、宁德、雅安、福泉）分别负责不同材料体系的生产和扩
产。

应用领域：产品广泛应用于3C 消费电子（手机、笔记本）、新能源汽车
（动力）、储能以及新兴领域（低空飞行、固态电池）。
2.
业务结构

收入占比：

钴酸锂：公司的“现金牛”业务，占据收入大头，全球市占率超
40%，在高端消费电池领域占据统治地位。

三元材料：主要应用于动力电池，是公司增长的第二曲线，特别是中
镍高电压路线在国内占据重要份额。

其他：包括磷酸铁锂（储能/动力）和氢能材料（镍氢电池）。

销售模式：以直销为主，主要客户包括宁德时代、中创新航、松下、TDK
等全球知名电池企业。
3.
产业链地位

上下游关系：处于产业链中游。上游依赖锂、钴、镍等矿产资源，下游绑
定全球头部电池厂及终端品牌。

定价权：在钴酸锂领域拥有极强的定价权，是行业标准的制定者；在三元
材料领域，通过“原材料成本+加工费”的模式锁定利润。

成本结构：原材料（锂、钴、镍）占成本比重极高，因此对上游资源的保
供控价能力是核心竞争力之一。
4.
股东背景

实际控制人：福建省财政厅（通过厦门钨业控股）。
