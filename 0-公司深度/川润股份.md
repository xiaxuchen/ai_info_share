一、基本情况及生意特性
1.1 公司概况
川润股份全称四川川润股份有限公司，2008年9月上市，深交所主板。公司起家于润滑液压设备制造，在风电液压
润滑系统领域做到国内第一，市占率超42%，海风样机覆盖率超80%。经过30余年发展，公司目前以流体控制技
术和智能能源装备两条线并行发展，产品覆盖风电、核电、储能、数据中心等领域。
控股股东为罗丽华（持股12.58%）和钟利钢（持股5.88%），夫妻为实际控制人。2025年三季报前十大流通股东
中，巴克莱银行新进，牛散郭磊等4位股东新进入榜，机构关注度有所提升。
1.2 业务结构
公司主营业务分为两大板块：
流体控制及系统解决方案：这是公司的传统基本盘，包括智能润滑系统、液压控制系统、冷却系统集成等。主要
应用于风电（液压润滑系统市占率超42%）、核电、重型机械等领域，客户包括GE、西门子等国际工业巨头。这
块业务2025年前三季度同比增速约5%左右，贡献公司大部分营收。
液冷温控及新能源装备：这是公司的战略增长方向，包括数据中心液冷系统（冷板式+浸没式）、储能PCS液冷系
统等。2025年6月完成液冷产线智能升级，年交付能力提升至1GW，推出全链条液冷解决方案。储能液冷累计交
付超1000套，数据中心液冷已在北美市场交付超200MW。
公司未在财报中单独披露液冷业务的精确收入占比，但管理层透露新能源流体系统产品（包含液冷相关）2025年
前三季度同比增长40.12%，液冷是增长最快的业务板块。
1.3 产业链地位
上游：主要是钢材、铝合金、铜材、电子元器件等原材料供应商，采购集中度不高，议价能力一般。
下游：风电整机厂（金风科技、明阳智能等）、储能集成商、数据中心运营商和IDC服务商。公司在风电领域的客
户黏性较强（流体系统一旦进入供应体系，更换成本高），但在液冷领域尚处于拓展期，客户关系还在建立中。
应付/应收方面：2025年中报显示应收账款占最近一年营收的比例高达78.76%，体量偏大。这说明公司在产业链
中的议价能力不算强，存在一定的回款压力。

二、核心逻辑及业务分析
2.1 市场空间
液冷数据中心市场
这是川润股份最大的故事所在。AI算力爆发驱动数据中心散热需求质变，GPU功耗从450W攀升至2700W（英伟
达GB200），传统风冷已达物理极限。
中国信通院测算：2024年中国智算中心液冷市场规模184亿元，同比增长66.1%，预计2029年达到约1300亿元，5
年增长约7倍。IDC数据：2024年中国液冷服务器市场规模23.7亿美元（约170亿人民币），同比增长67%。两个
数据口径不同但指向一致：市场在爆发式增长。
冷板式液冷是目前主流路线，IDC预计2030年将占液冷市场90%的份额。浸没式液冷散热效率更高但生态不成熟
，目前主要在特定超高密度场景应用。
渗透率方面，当前中国数据中心液冷渗透率约10%左右，预计2030年提升至30%-40%。政策端要求新建数据中心
PUE降至1.25以下（部分地方要求1.15），液冷是达成目标的核心手段。
风电液压润滑市场
中国"十五五"风电装机目标持续推进，海上风电是主要增量。川润在风电液压润滑系统领域市占率超42%，海风样
机覆盖率超80%，累计交付超20万台套。这块市场相对成熟，增速不高但稳定。
储能热管理市场
中国新型储能装机量持续高增，液冷方案在大型储能电站中的渗透率快速提升。但这个市场参与者众多，竞争激
烈，毛利率普遍不高。

需求端驱动
AI算力投资是中国当前最确定的基础设施投资方向之一。三大运营商、互联网巨头、地方政府都在建设智算中心
，对液冷的需求是确定性的。2025年中国智算算力预计达1037.3 EFLOPS（FP16），同比增长43%。
风电和储能的装机增长为传统业务提供稳定基本盘。
供给端驱动
川润2025年6月完成液冷产线升级，年交付能力提升至1GW。CDU、冷板、分集水器等核心产品已实现自主生产
，集装箱式液冷系统也已推出。3MW模块部署周期从90天缩短到45天，交付效率有提升。
2024年研发投入6200余万元，2025年上半年研发投入2300余万元，拥有液冷相关专利40余项。
盈利能力改善路径
从2025年的数据看，毛利率逐季回升：Q1为13.5%，上半年为16.75%，前三季度为17.8%。这个趋势如果持续，
说明产品结构在优化（液冷产品毛利率通常高于传统流体控制产品）。
三、主要风险
3.1 持续亏损风险
这是最现实的风险。公司2022-2025年连续四年盈利能力极度承压，2022年几乎零利润，2023和2024年合计亏损
约1.79亿元，2025年预计继续亏损1900-2700万元。这意味着公司已经连续四年没有创造股东价值。
亏损的原因是多方面的：传统业务毛利率下滑、液冷等新业务尚处于投入期（产线建设、研发投入、市场开拓都
需要花钱）、风电行业周期波动等。如果2026年不能实现扭亏，公司的投资逻辑会严重打折。
3.2 液冷竞争加剧
液冷赛道的热度已经吸引大量玩家涌入。英维克2024年液冷营收已经超过20亿元，申菱环境、高澜股份等都有成
熟的产品和客户体系。川润作为后来者，在品牌认知、客户资源、项目经验等方面都有明显差距。如果液冷市场
增速放缓或竞争进一步加剧，川润可能面临"起了个大早赶了个晚集"的局面。
3.3 应收账款风险
应收账款占最近一年营收的78.76%，这个比例偏高。公司客户包括风电整机厂、储能集成商、数据中心运营商等
，这些行业普遍存在回款周期长的问题。如果宏观经济下行或客户资金链紧张，应收账款可能面临减值风险。
3.4 股价与基本面脱节
截至2026年4月10日，公司总市值92.7亿元，PB 6.21倍。而公司2024年归母净资产约15.1亿元，2025年预计继续
微亏，净资产基本没有增长。以近93亿市值对应一家连续四年微利或亏损的公司，估值显然包含了很强的液冷预
期。一旦业绩兑现不及预期，估值回调空间很大。
3.5 技术路线不确定性
冷板式液冷和浸没式液冷哪条路线最终胜出还不确定。当前冷板式占主流，但浸没式在超高密度场景有天然优势
。如果行业技术路线发生重大变化，公司的产线和研发投入可能面临沉没成本风险。
四、其他重要问题
4.1 控股股东减持
2025年初，实控人之一罗丽华及部分董事、高管实施了减持计划，截至2025年2月已完成。控股股东减持本身就
是负面信号——如果管理层自己都不看好短期股价，外部投资者需要更加谨慎。
4.2 资本开支
2025年前三季度投资活动现金流净流出6302万元，主要用于液冷产线升级和产能扩张。公司目前的资产负债率约
58%（2025年三季报），不算特别高，但如果持续加大液冷投入，财务杠杆可能进一步上升。

五、初步价值评估与投资判断
5.1 核心投资逻辑
川润股份的估值逻辑不建立在过去业绩上（因为过去四年业绩很差），而是建立在一个假设上：AI算力爆发将推
动液冷市场爆发式增长，川润凭借30年工业液冷技术积累，能在数据中心液冷这个千亿市场中分到一杯羹，从而
实现营收和利润的量级跃升。
这个逻辑能不能兑现，取决于三个可验证的命题：
第一，液冷市场是否真的如预期高速增长？目前看，IDC和信通院的数据都指向高速增长，这个前提暂时成立。
第二，川润能否在竞争激烈的液冷市场中拿到足够订单？这是最关键的变量。2025年与香江科技的合作是积极信
号，但距离大规模订单落地还需要时间。
第三，公司能否实现盈利？连续四年亏损，说明转型阵痛比想象中更长。2025年全年预亏1900-2700万，全年营
收大约在17-18亿元，但仍无法覆盖所有成本。

六、最终结论
川润股份是一家正在经历业务转型的传统制造企业。传统流体控制业务稳定但增速有限，液冷业务方向正确但兑
现速度太慢。公司已经连续四年盈利能力极差，2025年预计继续亏损，意味着转型尚未完成。
当前92.7亿市值、6.21倍PB的估值，完全建立在"液冷业务将爆发"这个预期上。如果2026年不能实现有意义的盈
利（至少1亿元净利润以上），这个估值支撑会很脆弱。
