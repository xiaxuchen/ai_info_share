安集科技（688019）
报告时间：2026年5月1日 | 作者：开卷有财
一、基本情况及生意特性
1. 业务范围
安集科技（688019.SH）成立于2006年，2019年7月在科创板上市，是国内CMP抛光液龙头供应商。公司核心业务包括：
CMP抛光液（核心业务，高毛利率）：
• 半导体制造用化学机械抛光液
• 覆盖硅氧化物、硅氮化物、钨、铜、铝等多种材料抛光液
• 应用于逻辑芯片、存储芯片制造的全流程
电镀液及配套试剂：
• 晶圆制造及先进封装用电镀液
• 2026年开始实现量产销售，打开新增长曲线
湿电子化学品：
• 晶圆制造用清洗液、刻蚀后清洗液等
• 毛利率较抛光液低，但市场空间大
业务结构（2026Q1数据）：
• 总营收7.24亿元（单季度，同比增长32.76%）
• CMP抛光液占比约70%，电镀液及湿化学品占比约30%
• 公司战略：巩固CMP抛光液龙头地位，拓展电镀液新业务
2. 产业链地位
上游：化工原材料供应商，部分关键原料（如研磨颗粒）依赖进口
下游：晶圆制造企业（中芯国际、华虹半导体、长江存储、台积电、格芯等）、先进封装企业
议价能力分析：
• 应收账款周转天数：约85天（2026Q1），较上海新阳更短，对下游议价能力更强
• 应付账款周转天数：约65天，对上游议价能力一般
• 预付账款较少，供应商关系稳定
成本结构（2026Q1）：
• 营业成本3.22亿元，占营收44.5%
• 毛利率55.5%，显著高于上海新阳（42.2%）
• 研发费用1.19亿元，占营收16.4%，研发投入强度更高
3. 股东背景
• 实际控制人：Shumin Wang（美籍华人，公司创始人）
• 机构持股：2026Q1显示多家公募基金增持，北向资金持仓增加
• 股权激励：2025年推出股权激励计划，解锁条件与营收、研发进度挂钩
4. 财务特征
分红/融资历史：
• 上市以来累计分红约2.8亿元
• 2019年IPO募资约15亿元，2022年定增募资约12亿元，用于CMP抛光液及电镀液项目扩产
• 2026年无新增融资计划
资产减值情况（2026Q1）：
• 存货11.24亿元，存货周转天数约95天，略高于上海新阳
• 应收账款6.18亿元，占营收比例较高，需关注回款风险
• 商誉0（无并购形成的商誉）
运营资本周期：
• 经营性现金流净额：2026Q1未披露具体数值，但2025年全年为4.75亿元，转正
• 自由现金流-2,707万元，主要因扩产项目投资增加

• 电镀液业务与CMP抛光液共享客户资源，协同效应显著
3. 成长驱动
产能扩张计划：
• 上海基地扩产项目：2026年产能预计增长40-50%
• 安徽项目：2027年投产，产能再次提升50%以上
• 产能释放节奏：2026年产能利用率约70-80%，2027年满产
盈利提升路径：
• 产品结构优化：高毛利率的先进制程抛光液占比提升，预计2026年综合毛利率提升至56-58%
• 规模效应：产能利用率提升后，单位固定成本下降
• 电镀液业务放量：2026年开始量产销售，毛利率预计55%+，将显著提升综合毛利率
新产品与新市场：
• 先进制程用CMP抛光液：7nm及以下产品预计2026年下半年通过客户认证，2027年量产
• 电镀液：2026年开始量产销售，预计2026年营收贡献5,000-8,000万元，2027年可达2-3亿元
• 海外市场：2025年海外营收占比约12%，目标2027年提升至20%+
三、主要风险
1. 行业风险
• 国产替代进度不及预期：若下游晶圆厂扩产放缓或国产化率提升速度低于预期，公司产能利用率将受影响。量化影响：若行业需求增速
从15%降至5%，公司营收增速可能从30%降至15-20%。
• 技术路线颠覆：若CMP工艺被新技术（如 atomic layer deposition、选择性沉积等）替代，公司核心产品需求将受冲击。目前来看，CM
P工艺在7nm及以下制程仍是主流，技术路线颠覆风险较低，但需持续跟踪。
• 行业周期性波动：半导体行业具有周期性，若全球半导体景气度下行，下游晶圆厂资本开支缩减，将影响公司订单。
2. 经营风险
• 产能落地延迟：安徽项目若进度不及预期，将影响2027年产能释放。目前项目按计划推进，延迟风险较低。
• 关键研发失败：7nm及以下先进制程用CMP抛光液若认证失败，将影响长期成长逻辑。目前公司产品已在客户验证中，失败风险可控。
• 大客户流失：若中芯国际、台积电等核心客户削减订单或切换供应商，将对公司业绩造成显著冲击。量化影响：若前五大客户流失任意
两家，营收可能下降25-35%。
