宏达电子（300726.SZ）
一、公司基本情况
宏达电子于2017年11月在深交所创业板上市，是国内军用钽电容器生产领域的龙头企业。公司以钽电容器
为起点，逐步拓展至多层瓷介电容、陶瓷薄膜电路、电源模块等电子元器件和模块业务，始终聚焦高可靠
领域国产化需求。
公司最新收盘价58.81元（2026年5月14日），总市值约245亿元。
2025年度，公司实现营收18.87亿元，同比增长33%；归属于上市公司股东的净利润4.85亿元，同比增长55
%。扣非净利润增速更高，显示出主营业务盈利能力的持续改善。

2.3 生意特性分析
高毛利率： 电子元器件业务毛利率普遍在60%以上，盈利能力突出。
军工属性： 高可靠（军工）领域是公司核心市场，客户粘性高、订单稳定性强。
品类扩张： 公司持续拓展产品线，从钽电容延伸至MLCC、电源模块、陶瓷电路等，形成平台效应。
民品提速： 公司加大民品市场开拓，2025年民品收入3.40亿元，同比增长59%，成为新增长极。
三、核心投资逻辑
3.1 军工电子：高可靠需求持续增长
军工电子是公司的传统优势领域：
• 武器装备信息化升级带动电子元器件需求
• 国产替代加速，军工供应链向国内厂商倾斜
• 公司是多家军工院所的合格供应商，订单稳定性强
3.2 民品市场：光模块等新赛道快速放量
民品业务是公司最重要的增量方向：
• 公司相关产品已成为多家头部光模块厂商的合格供应商
• 单层瓷介电容、陶瓷薄膜电路等民品收入同比增长95%
• AI驱动光模块需求爆发，公司直接受益
光模块用电子元器件技术门槛高、认证周期长，公司先发优势明显。
3.3 平台化战略：品类扩张提升成长天花板
公司从单一钽电容厂商转型为电子元器件平台：
• 电源模块营收3.33亿元，同比增长27%，成为第二大业务
• MLCC、陶瓷电路等新品持续放量
• "产品+解决方案"模式提升客户粘性和单客户价值量
品类扩张打破钽电容的天花板限制，为长期成长打开空间。

## 关键要点

**投资逻辑**: 从"钽电容龙头"转型为"高可靠电子元器件平台"，军工+民品双轮驱动；民品光模块配套受益AI爆发（2025年民品收入3.40亿+59%），品类扩张（MLCC/电源模块/陶瓷电路）打开成长天花板。

**核心竞争力**: 军用钽电容龙头（毛利率60-70%），军工合格供应商客户粘性高，已进入多家头部光模块厂商合格供应商体系，非钽业务占比达66%完成平台化转型。

**主要风险**: 军工需求受国防预算和采购周期波动影响，民品市场竞争激烈价格压力大，对军工大客户依赖度较高，PE约50x相对增速估值偏高。

**财务亮点**: 2025年营收18.87亿(+33%)、归母净利4.85亿(+55%)，扣非增速更高，综合毛利率60%盈利能力突出，民品光模块配套收入同比增95%。

**估值参考**: 当前PE(TTM)约50x、股价58.81元接近中性下限；中性（2027净利5.5-6.5亿）合理市值275-325亿/65-80元；悲观200-250亿/48-60元；乐观350-400亿/85-95元。
