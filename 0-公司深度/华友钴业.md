一、基本情况及生意特性
1.1 公司概况
华友钴业（SH 
603799），2002年创立于浙江桐乡，2015年1月登陆上交所主板。公司最初是一家专注钴产品冶
炼的"小金属"企业，如今已演变为全球规模最大的三元前驱体生产商之一，同时是刚果（金）钴矿最重要的中国买
家，以及在印尼拥有规模最大的中国镍湿法冶炼产能的企业。
实控人：陈雪华，华友控股集团董事长，持有华友控股100%股权。华友控股为上市公司第一大股东，持股16.27
%；陈雪华本人直接持股4.35%，合计控制20.62%股份。第二大股东为香港中央结算，持股9.30%（代表外资通
过沪深港通持有）。十大股东中其余均为ETF基金，说明机构主要通过被动指数方式持仓。
主营业务：公司将业务分为三大板块：
• 锂电材料：三元前驱体（NCA/NCM）、正极材料（LFP及三元）、钠电前驱体等。这是营收占比最大的板块，
亦是公司向终端新能源供应链靠近的核心业务。
• 能源金属：钴产品（包括粗制氢氧化钴、电解钴、钴盐等）、铜产品、镍中间品（MHP/高冰镍）、碳酸锂等原
材料。这是公司传统优势，同时也是利润率最高的板块。
• 能源材料：铜箔、锂电铝箔等。这是延伸产业链的相对新兴业务。

二、核心逻辑及业务分析
2.1 业务特性：原材料属性与加工属性共存
华友钴业是一门容易被误解的生意。表面上看，它是三元前驱体加工商，毛利率只有17%，看上去像个普通制造
企业。但实际上公司有两类截然不同的资产：
资源型资产（高毛利、强周期）：非洲钴铜矿、印尼镍湿法项目。这些资产的核心竞争力是占据了资源地，一旦
建成就有稳定的成本优势。钴产品2025年毛利率高达36.78%，同比提升21.24个百分点，主要就是钴价回升的贡
献。
加工型资产（低毛利、稳定）：三元前驱体和正极材料。毛利率在10-15%之间，相当于代工加工。好处是出货量
大、与大客户深度绑定，坏处是毛利随加工费用变化而波动，受金属价格影响很大。

2.3 市场空间与竞争格局
三元前驱体市场：
三元前驱体的需求直接依附于三元锂电池的装机量。2025年全球三元材料产量约103.3万吨，同比增长7.4%（而
磷酸铁锂产量同期增长63%）。三元路线在乘用车高端市场（高续航、高能量密度需求）仍有明显优势，在大圆
柱电池（特斯拉4680、国内46系电池）和固态电池路线中依然是主流选择。

竞争格局方面，三元前驱体行业前三名是中伟股份、格林美、华友钴业，三者合计市占率超过60%。其中：
• 中伟股份：总出货量第一，印尼有镍资源布局，出货均衡但高端占比低于华友
• 格林美：循环利用起家，综合出货量大，但高端超高镍占比不如华友
• 华友钴业：在超高镍（9系）市场份额超33%位居第一，高端动力圆柱三元材料市占率超60%。与LG新能源签有
7.6万吨前驱体长单，锁定了特斯拉、宝马、大众等高端整车客户供应链。
华友的差异化策略是"高端化"——不打低价战，而是专注超高镍、大圆柱等技术门槛较高的产品，通过与车企直接
合作锁单降低价格不确定性。2025年完成46系大圆柱电池前驱体开发，并切入固态电池供应链，是公司持续提高
技术壁垒的体现。
钴业务：
华友在刚果（金）拥有华圭、华星、华汇等多个铜钴矿。钴是动力电池和消费电子的核心原料，中国企业在刚果
（金）钴矿的采购和冶炼环节占据了全球60%以上的份额，华友是其中规模最大的玩家之一。
2025年是钴业的"特殊年份"：刚果金2025年2月起实施钴出口禁令（后改为配额制），导致钴价从低位快速拉升。
华友2025年钴产品毛利率高达36.78%，同比提升21.24个百分点，是当年业绩超预期的最重要原因。但需要注意
，出口禁令和配额政策具有不确定性，后续政策变化可能影响钴价。
印尼镍业务（最重要的增量来源）：
这是华友未来3-5年最重要的利润增量点，也是公司最大的投资亮点之一。
华友在印尼的镍资源布局非常系统：
• 华越项目：华友100%控股，已满产，采用HPAL湿法冶炼，产品为MHP（氢氧化镍钴）
• 华飞项目：已投产放量
• 华科项目：已投产
• 华杉项目：已全线投产，处于产能爬坡阶段
• Pomalaa项目：与淡水河谷印尼、福特汽车合作，年产12万吨镍金属量，2026年内建成
• Sorowako项目：与淡水河谷印尼合作，年产6万吨镍金属量，2026年启动建设
2025年印尼MHP出货量达23.65万吨，同比增长约58.72%。公司累计锁定印尼镍资源超14亿湿吨，资源储量极为
充足。
印尼镍业务的逻辑非常清晰：通过控制上游镍资源，公司在三元前驱体的原料端实现自供，将外采镍的成本优势
内部化。自2022年大规模投入资本开支以来，印尼镍项目从无到有，现在逐步从资金消耗期进入利润贡献期。东
吴证券预测，到2026年，仅镍业务一项就可贡献净利润超30亿元。
2.4 成长驱动
印尼镍产能持续释放
目前华越等在产项目证明了技术可行性和盈利能力。Pomalaa 12万吨大项目2026年投产后，公司镍金属产出将再
上一个台阶，边际成本还将进一步降低，前驱体镍自供率提升直接改善盈利质量。这是确定性最高的增量来源，
也是为什么东吴等券商对2026年业绩如此乐观的主要依据。
钴价能否维持高位
2025年钴价上涨的直接原因是刚果金的政策干预。即便未来政策趋于稳定，2022-2024年钴矿投入不足导致的供
给缺口仍将持续一段时间，钴价回到2023年底部（约20万元/吨）的可能性较低。但这个判断有不确定性，政策是
刚果金说了算的。
超高镍三元的技术护城河
大圆柱电池（46系）的量产推进、固态电池商业化进程，都指向超高镍三元材料需求的继续增长。华友在9系超高
镍33%的市占率、高端圆柱三元超60%的市占率，是长期积累的结果，短期难以被竞争对手撼动。但前提是三元
路线整体不被铁锂彻底边缘化，这个前提目前看仍然成立，高端市场暂时稳固。

3.1 金属价格波动是最大的不确定性

华友的利润对钴、镍、锂价格高度敏感。2023年就是反例：营收+5%但净利润下降14%，完全因为金属价格下跌
侵蚀了利润。钴价当前处于政策扰动后的高位，一旦刚果金政策松动或全球供给恢复，钴价可能回调。镍价则长
期受印尼供给过剩压制，2025年虽然因配额政策出现短期反弹，但全球镍供给仍处于过剩格局。
3.2 三元路线面临磷酸铁锂替代压力
磷酸铁锂在中低端车型的渗透率已超过60%，2025年LFP产量同比增长63%，远超三元7.4%的增速。如果LFP的
低成本优势进一步向中高端车型扩展（比亚迪的刀片电池已用于汉、唐等中高端车型），三元材料的市场空间将
被压缩。这个趋势在2023年已经直接影响了华友的业绩。
公司的应对策略是聚焦超高镍+大圆柱+固态电池，走差异化路线，但市场总量的增速确实慢于LFP，这是结构性
压力。
3.3 海外政策风险
华友的核心资产分散在刚果（金）和印尼。刚果金政治局势不稳定，矿业税收和出口政策调整频繁（2025年出口
禁令是近年最典型的案例）；印尼的RKAB镍矿配额制度每年都存在不确定性，曾多次造成市场恐慌。这两个国家
的政策风险是无法完全对冲的系统性风险。
3.4 重资产扩张导致财务杠杆偏高
当前资产负债率62%、净有息债务254亿，虽然在重资产行业尚可接受，但如果金属价格出现系统性下行（2023
年就发生过），公司同时面临盈利下滑和融资收紧的双重压力，财务弹性会显著降低。连续5年自由现金流为负，
分红能力受限，也意味着股东回报更多依赖股价而非现金分红。
3.5 实控人质押比例较高
陈雪华累计质押股份超40%，这是一个需要关注的股权风险。如果股价持续下行，质押可能触发强制平仓，对股
价产生负面反馈。

四、其他重要问题
4.1 为什么业绩创新高但股价仍在历史高点一半位置？
这个问题值得深思。2025年净利润61亿，比2021年的39亿多了57%，但股价只有2021年高点116元的53%。
原因有几个：
1. 2021年市场给华友的估值逻辑是"锂电新能源主题"，享受了主题溢价（彼时PE一度超过60倍）。现在市场把华
友更多看成有色金属资源股，给的是商品周期类估值（PE 15-20倍），估值中枢系统性下移。
2. 2021年后三元vs铁锂的路线之争让市场对华友的增长空间产生了实质性疑虑，这个疑虑至今没有消散。
3. 高负债、持续为负的自由现金流、低分红，使得现金流估值法对华友并不友好。
换句话说，股价相对历史高点的折价，部分是合理的估值回归，而非纯粹的"被低估"。
4.2 分红政策
2025年分红比例仅15.52%，每股分红约0.50元，对应当前股价股息率约0.8%，不是靠分红吸引人的股票。公司
称处于成长期、资源开发投入大，这个理由本身是真实的，但对股东而言意味着短期内不会有明显的现金回报。
4.3 东吴证券的乐观预测
东吴证券预计2026年华友归母净利润约104亿元，较2025年的61亿增长70%，对应当前1165亿市值的PE约11倍
，确实非常便宜。这个预测基于的假设是：镍业务（以Pomalaa项目为主）、钴业务、碳酸锂业务各贡献约30亿
利润。这是非常乐观的情景——需要钴价维持高位、Pomalaa项目按时投产且达产顺利、碳酸锂价格不再继续下
行。
