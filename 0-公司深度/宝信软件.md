一、基本情况及生意特性
1.1 公司概况
宝信软件成立于1994年3月11日，是国内领先的工业软件和数据中心服务提供商。公司实际控制人为中国宝
武钢铁集团（央企背景），持股比例约50.6%。背靠国内最大钢铁集团，宝信软件在工业软件领域具备天然
的实践场景和客户资源。
主营业务分为两大板块：
• 工业软件及服务：面向钢铁为主的制造业，提供MES（制造执行系统）、ERP、行业解决方案等工业
软件产品
• IDC（数据中心）业务：宝之云品牌，在上海、北京、深圳等地运营多个数据中心，为大型互联网企业
、云计算厂商提供托管服务
1.2 业务结构
根据2025年报数据，公司营收结构大致如下：
• 工业软件及服务：约55-60%（含关联交易）
• IDC业务：约40-45%（宝之云数据中心）
关联交易占比较高（约30-35%），主要来自宝武集团体系内业务，这是央企背景公司的典型特征。
1.3 财务特征
核心财务指标（2025年年报）：
2026年一季报：
• 营收：26.44亿元
• 归属净利润：3.85亿元
• ROE：3.39%
净利润下滑主要原因是2024年同期有较大基数（2024年净利润23.27亿元），且钢铁行业信息化需求在202
5年有所承压。
现金流：公司经营现金流整体稳定，2025年经营现金流净额约21.47亿元，现金流质量良好。分红率长期维
持在40-50%水平，在软件类公司中属于较高水平。
资产质量：资产负债率约46%，流动比率1.84，财务结构稳健。无重大商誉减值风险。

1.4 产业链地位
公司在钢铁行业信息化领域处于龙头地位，MES产品在钢铁行业市场份额领先。IDC业务凭借宝武集团在上
海等核心区域的能耗指标和土地资源，具备一定稀缺性。客户涵盖腾讯、阿里、字节跳动等头部互联网企
业。

二、核心逻辑及业务分析
2.1 市场空间
工业软件市场：
• 国内MES市场空间约300-400亿元，钢铁行业信息化需求与钢价周期、下游投资相关
• 宝信软件深耕钢铁行业30年，具备明显先发优势和客户粘性
IDC市场：
• 国内数据中心市场规模超过3000亿元，受益于AI算力需求爆发式增长
• 核心区域（北上广深）能耗指标稀缺，宝信的宝之云在上海及长三角区域布局较早
• AI大模型训练带来算力需求爆发，头部互联网客户的IDC需求持续增长
2.2 竞争格局
工业软件领域：
• 国内钢铁行业信息化主要玩家包括宝信软件、中冶赛迪、首自信等
• 宝信软件背靠宝武，在宝武体系内市场份额绝对领先
• 向其他钢铁企业（如沙钢、建龙等）拓展时面临一定竞争
IDC领域：
• 第一梯队：阿里云、腾讯云、华为云等自建数据中心
• 第二梯队：宝信软件、万国数据、光环新网等第三方IDC厂商
• 宝之云的优势在于核心区域资源禀赋和央企信用背书
2.3 成长驱动
短期催化因素：
1. AI算力需求爆发：腾讯、字节等大客户对IDC机柜需求持续增长，宝之云上架率有望提升
2. 宝武集团数字化转型：作为央企数字化转型标杆，宝武持续投入信息化升级
3. 宝之云四期等新建数据中心逐步投产，贡献增量收入

三、主要风险
3.1 行业风险
钢铁行业周期性风险：宝信软件约30%以上收入来自宝武集团关联交易，钢铁行业景气度直接影响信息化
投入。2025年钢铁行业整体承压，导致工业软件业务增速放缓。
IDC行业竞争加剧：万国数据、光环新网等竞争对手持续扩张，头部互联网厂商自建数据中心趋势明显，可
能影响第三方IDC厂商的市场空间。
3.2 经营风险
关联交易依赖：对宝武集团依赖度较高，如果宝武大幅削减IT投资，短期内对业绩影响较大。
IDC上架率风险：新建数据中心从投产到满租需要1-3年爬坡期，若大客户需求不及预期，可能导致机柜利
用率偏低，影响盈利能力。
应收账款回收：2025年末应收账款约20亿元，占营收比例约18%，需关注回款节奏。
3.3 财务风险
毛利率下降压力：IDC业务毛利率低于软件业务，随着IDC收入占比提升，整体毛利率存在下降压力。
研发投入持续性：工业软件需要持续研发投入，若研发效率下降，可能影响产品竞争力。

## 关键要点

- **投资逻辑**：央企背景工业软件+IDC双轮驱动，受益宝武集团数字化转型和AI算力需求爆发，稳定现金流+高分红提供防御属性。

- **财务亮点**：经营现金流净额21.47亿元质量良好、分红率维持40-50%、财务结构稳健（资产负债率46%）。
