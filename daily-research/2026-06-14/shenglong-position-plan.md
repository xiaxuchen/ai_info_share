## 三、建仓区分析

### 核心原则

当前价格 27.74 元处于二连板涨停状态，**一字涨停无有效换手**，筹码密集区在 22-24 元。**严禁涨停板追入**，需等待首板放量后的自然回踩。

### 第一建仓区：23.00 - 24.00 元 ⭐⭐⭐⭐⭐（建议仓位 8-10%）

| 维度 | 分析 |
|------|------|
| **逻辑** | 回踩首板（6月10日）涨停板起点22.93元附近 |
| **支撑** | 22.93元 = 首板启动前最后一根K线收盘价，突破后转化为支撑 |
| **均线** | 20日均线约在 23.50 元附近，构成中期支撑带 |
| **仓位** | 总资金的 **8-10%**（单只标的常规配置） |
| **触发条件** | 二连板开板后回调至 24 元以下，分3次低吸（24.00/23.50/23.00） |

### 第二建仓区：21.50 - 22.50 元 ⭐⭐⭐⭐（补仓 5%）

| 维度 | 分析 |
|------|------|
| **逻辑** | 4月低点区间（21.43-22.15元），历史成本密集区 |
| **风险** | 若跌破22.93首板起点，说明以钼代钨主题未获充分认可 |
| **仓位** | 额外加仓 **5%**，总仓位升至 **13-15%** |
| **触发条件** | 深度回调至22.50以下，且**成交量缩至5日均量50%以下** |

### 禁止介入区：27.00 元以上 ❌

- 二连板后溢价严重，追入即承担15-20%回调风险
- 一字板无换手，一旦开板容易出现跌停式回调

## 五、仓位管理原则

| 原则 | 内容 |
|------|------|
| **单只上限** | 总资金的 **15%**（含补仓） |
| **分步建仓** | 绝不全仓一次性买入，至少分3次 |
| **回撤控制** | 单票浮亏 >8% 强制减半仓 |
| **盈利加仓** | 首仓盈利 >15% 且技术面确认后，可考虑金字塔加仓 |
| **联动监控** | 同时关注金钼股份（市值过大）、欧莱新材（同主线）的走势联动 |
